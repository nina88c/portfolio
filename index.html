Weather Project

It allows users to search for weather conditions in a specific city or retrieve weather information based on their current location. The website displays the city name, current temperature, weather description, humidity, wind speed, and date/time. It also provides a weather forecast for the upcoming days. World Clock Project

This website is a World Clock application that allows users to view the current time and date in different cities around the world. Users can select a city from the provided options or choose their current location. The website dynamically updates the displayed time and date for the selected city, and the time is continuously refreshed. The design features a world map background and a container with city options and their corresponding time and date information.

Hobby Project

This website is a hobby project that focuses on UX Design. It provides information about UX design, its process, and its goal of creating meaningful user experiences. The website offers a free UX course to the first 50 sign-ups and includes a button to apply. It dynamically updates the heading based on the user's age. The website aims to educate visitors about UX design and encourage them to enroll in the course.
